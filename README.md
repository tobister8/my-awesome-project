# Prediction Market Monitoring Dashboard

A real-time web application built with Streamlit to track, analyze, and visualize data from prediction markets. This dashboard provides a clean interface for monitoring shifting odds, implied probabilities, and market technicals across various categories.

---

## 🚀 Features

* **Real-Time Data Tracking:** Fetches live market data and updates probabilities dynamically.
* **Interactive Visualizations:** Uses clean charts to track sentiment shifts and historical trends.
* **Filtering & Search:** Easily sort through different markets, volumes, and categories.
* **Technical Insights:** Displays implied probabilities alongside raw market prices for quick analysis.

---

## 🛠️ Tech Stack

* **Frontend/Framework:** [Streamlit](https://streamlit.io/)
* **Language:** Python 3.9+
* **Data Handling:** Pandas, NumPy
* **APIs/Data Sources:** Prediction market REST APIs / WebSockets
